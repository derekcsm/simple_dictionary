Android dictionary app that is easy to use and features TTS. Built on the Wordnik API


Libraries used in this project:

- Jake Wharton/ Actionbar Sherlock

- BeardedHem/ Android Bootstrap

- tills13/ Define

- Wordnik API

- Jazzy List View

- Nine Old Androids

- Google Play Services
